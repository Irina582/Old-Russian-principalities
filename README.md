# Домашнее задание

## Задание
Задание: Работа с коллекциями, функциями, классами.
Домашнее задание состоит из 2 частей. В первой части ДЗ необходимо внедрить 2 задания по варианту в приложение вашей третьей лабораторной. Например, функция поиска палиндромов используется для выбора покупателей для подарков (тема Система электронных покупок).

Нужно использовать данные по вашей теме (текст, товары и тд), а переменные/функции в коде должны называться по этой теме (account, account_number и тд). Обязательно нужно использовать:

1) цикл с (пост)условием, но не по счетчику (например после 12:00 или после нужного сообщения)
2) строка
3) объект
4) хотя бы 1 коллекцию (массив или множества)

Во второй части ДЗ необходимо на странице Подробнее выводить вместе с картинкой 3D модель. Модель .glb можно использовать одну и ту же для всех Услуг, но она должна быть по теме вашей работы
